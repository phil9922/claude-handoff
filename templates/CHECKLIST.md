# Checklist

> Format matters — the hooks parse these lines:
>
>     - [ ] [45m] Specific, actionable task
>
> The bracketed estimate is required: minutes (`30m`) or hours (`2h`, `1.5h`).
> Tick a box (`- [x]`) and the handoff updates itself.
>
> Optional: add a success criterion as a sub-bullet, and the box can't be called
> done until it's actually been checked this session:
>
>     - [ ] [45m] Wire auth middleware to session store
>       - ✓ login survives a server restart (integration test passes)
>
> The `✓` marker also accepts `v:` or `verify:`.
